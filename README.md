# Ecommerce Microservice

## Usage

Install pnpm:

```bash
npm install -g pnpm
```

To install all packages for all workspaces:

```bash
pnpm install
```

Run all in dev mode:

```bash
pnpm dev
```

## Run Docker Compose

```sh
docker compose up -d
```

## Create new service using Hono & Nodejs

```sh
pnpm create hono <my-app>
```

Select nodejs from framework
